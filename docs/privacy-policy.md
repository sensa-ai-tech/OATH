# Oath 隱私權政策（草稿）

> 最後更新：2026-02-13
> 狀態：Phase 0 草稿 — Phase 4 需由法律顧問審閱後定稿

---

## 一、蒐集機關名稱

[公司/個人名稱]（以下簡稱「本服務」）

## 二、蒐集目的

本服務蒐集個人資料之目的為提供命理分析與正向引導服務，包括：
- 計算個人命盤（西洋占星 + 八字命理）
- 生成每日個人化運勢與行動建議
- 提供互動式命理指引
- 改善服務品質與使用者體驗

## 三、個資類別

本服務蒐集的個人資料包括：
- **帳號資料**：電子郵件地址
- **出生資料**：出生日期與時間、出生地經緯度、性別
- **使用行為資料**：App 使用頻率、功能使用記錄（匿名化）
- **裝置資訊**：裝置類型、作業系統版本（匿名化）

本服務**不蒐集**以下資料：
- 真實姓名（非必要）
- 身分證字號或其他政府核發的身份證件
- 財務資訊（付款由 Apple/Google 平台處理）
- 通訊錄或社交網路資料

## 四、利用期間

- 帳號存續期間
- 帳號刪除後 30 天冷卻期（期間可恢復帳號）
- 冷卻期結束後永久刪除所有個人資料

## 五、利用地區

- 中華民國境內
- Supabase 雲端伺服器所在地（美國 AWS）
- Anthropic API 伺服器（美國）— 僅傳輸匿名化的命理計算結果文字，不含個人識別資訊

## 六、利用對象

本服務營運團隊。本服務不會將您的個人資料出售或提供予第三方。

## 七、利用方式

以電子方式處理與利用，所有資料以加密方式儲存（AES-256）。

## 八、當事人權利

根據個人資料保護法，您享有以下權利：
1. **查閱權**：查閱您的個人資料
2. **複製權**：取得您的個人資料副本
3. **補充更正權**：更正不正確的個人資料
4. **停止蒐集利用權**：要求停止蒐集、處理或利用您的個人資料
5. **刪除權**：要求刪除您的個人資料

行使上述權利，請透過 App 內設定頁面操作，或來信至 [聯繫信箱]。

## 九、跨境傳輸

本服務使用以下第三方服務，可能涉及跨境資料傳輸：

| 服務 | 伺服器位置 | 傳輸資料 | 說明 |
|------|-----------|---------|------|
| Supabase | 美國 (AWS) | 帳號 + 出生資料 + 運勢 | 加密儲存 |
| Anthropic (Claude API) | 美國 | 匿名化文字 | 不含個人識別資訊 |
| Sentry | 美國 | 錯誤報告 | 不含個人資料 |
| PostHog | 歐盟/美國 | 匿名化使用行為 | 無法追溯個人 |

## 十、資料安全措施

- 所有資料傳輸使用 HTTPS/TLS 加密
- 資料庫層級加密（AES-256）
- Row Level Security (RLS) 確保使用者只能存取自己的資料
- 定期安全稽核

## 十一、兒童隱私

本服務不針對 16 歲以下兒童提供服務。若發現未成年人使用本服務，將立即刪除相關資料。

## 十二、政策變更

本隱私權政策如有變更，將透過 App 內通知及電子郵件告知。重大變更將提供 30 天的審閱期。
