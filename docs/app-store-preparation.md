## 審核風險分析

| 風險 | Apple Guideline | 嚴重度 | 緩解措施 |
|------|----------------|--------|----------|
| 被歸類為算命/占卜 | 4.3 (Spam) | 高 | 強調正向心理學差異化 |
| 隱私問題 | 5.2.5 | 中 | 完整隱私政策 + 帳號刪除 |
| IAP 合規 | 3.1.1 | 低 | 使用 RevenueCat + Apple IAP |
| 內容品質 | 4.0 | 中 | 專業命理內容 + AI 品質控制 |

## App Store 描述策略

### 標題
Oath — 每日正向指引

### 副標題
融合中西智慧的個人化成長夥伴

### 關鍵字
正向心理、每日指引、個人成長、星座、命理、自我提升

### 描述重點
- 強調「正向心理學引導」而非「算命」
- 強調「個人化成長建議」而非「預測未來」
- 提及「專業心理學研究支持的正向重框架技術」
- 包含免責聲明：「本應用程式提供的內容僅供參考」

### 避免用詞
- ❌ 算命、占卜、預測
- ❌ 保證、一定、絕對
- ✅ 正向指引、個人化建議、成長夥伴
- ✅ 文化智慧、命理分析（作為參考框架）

## Plan B：PWA 方案

若 App Store 審核被拒：
1. Web App 作為主要發行管道
2. 加入 PWA manifest + Service Worker
3. 引導使用者「加到主畫面」
4. 推播通知改用 Web Push API

## Apple 隱私營養標籤

### 收集的資料
- **聯絡資訊**：電子郵件
- **使用者內容**：出生日期時間、出生地
- **使用狀況資料**：產品互動（匿名化）
- **診斷**：Crash 資料、效能資料

### 未收集的資料
- 健康與體適能
- 財務資訊
- 位置（僅註冊時一次性提供出生地）
- 通訊錄
- 瀏覽歷史
- 搜尋歷史
