# 決策記錄 001: Swiss Ephemeris 授權

> 狀態：待決策
> 日期：2026-02-13

## 背景

Swiss Ephemeris 是西洋占星計算的事實標準函式庫，提供精確的行星位置計算。

## 選項

### 選項 A：AGPL 授權（免費）
- **優點**：零成本
- **缺點**：整個專案必須開源（AGPL 傳染性）
- **影響**：商業模式可行但程式碼需公開

### 選項 B：商業授權（~$500 一次性）
- **優點**：程式碼可閉源、無授權風險
- **缺點**：前期成本
- **影響**：最安全的商業運營選擇

### 選項 C：Moshier 內建星曆表（免費、無授權限制）
- **優點**：零成本、無授權風險
- **缺點**：精度略低（但對大眾命理應用足夠）
- **影響**：需驗證精度是否符合需求（太陽/月亮位置誤差 < 1弧分）

## 評估標準

1. 精度需求（對於大眾命理 App 是否足夠）
2. 授權風險（是否影響商業模式）
3. 成本效益
4. 維護負擔
