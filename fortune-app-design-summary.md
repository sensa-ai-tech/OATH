# 中西合璧算命應用程式 — 設計概念摘要

## 願景

打造一款融合東西方命理體系的算命應用程式，以正向心理學為核心引擎，透過占卜的文化框架引導使用者產生自我實現預言（Self-fulfilling Prophecy），在不刻意透露意圖的前提下，幫助使用者建立正向心態與行為模式，催生更健康的社會氛圍。

---

## 核心設計哲學

- **Placebo Framing**：利用人類對命理系統的天然信任感，包裝正向心理學的引導內容
- **與詐騙的本質區別**：詐騙 App 掏空錢包，本產品充實使用者的心理資本
- **自我實現預言機制**：當使用者相信「命理預測」指向正面結果時，會無意識地調整行為趨向成功

---

## 命理體系架構

### 西方體系

| 體系 | 用途 | 受眾優勢 |
|------|------|----------|
| 占星術（太陽/月亮/上升星座、行星相位） | 每日運勢、個人命盤 | 年輕族群接受度高 |
| 塔羅牌 | 每日互動抽牌指引 | 視覺吸引力強 |
| 數秘術（Numerology） | 性格分析、流年運勢 | 簡單易懂 |

### 東方體系

| 體系 | 用途 | 受眾優勢 |
|------|------|----------|
| 八字命理 | 個人命盤、流年流月分析 | 台灣最普及的命理系統 |
| 紫微斗數 | 深度命盤解析 | 專業感強、信任度高 |
| 易經 | 每日一卦互動指引 | 文化深度、哲學感 |
| 農民曆宜忌 | 每日生活參考 | 在地文化共鳴 |

### 融合策略：統一解讀層

中西方體系各自運算後，由 AI 引擎將結果合成為一段統一、正向的解讀訊息。當兩套系統指向相似方向時，使用者會感受到「中西方都這麼說，一定很準」的強化效果。

> **範例**：火星與金星成四分相 ＋ 八字流日偏財受剋 → 「今天在人際與財務上可能有些張力，但這正是練習耐心溝通的好機會。試著在做重要決定前先深呼吸三次。」

## 內容引擎設計（三層架構）

```
第一層：可信的命理框架
  └─ 認真呈現每套命理系統的專業度，建立使用者信任

第二層：正向引導語言模型
  └─ 所有解讀遵循三原則：
     ├─ 承認困難但強調可掌控性
     ├─ 給予具體可執行的小行動
     └─ 將挑戰重新框架為成長機會

第三層：漸進式深度
  └─ 表面：每日運勢
  └─ 中層：個人命盤分析
  └─ 深層：目標設定、情緒覺察、冥想練習（以命理語言包裝）
```

## 成長槓桿

### 社交傳播

- 設計精美的每日運勢卡片，可一鍵分享至 IG Stories / LINE
- 算命天然具有高分享慾望，低成本高擴散

### 個人化體驗

- 基於生辰命盤，每天的內容都與「你」有關
- AI 確保每個使用者看到獨一無二的內容
- 產生「被理解」的感受，提高留存率

### 儀式感設計

- 每日固定時間推送「今日一卦」
- 搭配動畫與音效，讓開啟 App 成為令人期待的小儀式

## 技術架構

### 技術棧

- **前端**：Next.js（Web）/ React Native（App）
- **後端**：Supabase（資料庫 + Auth + Realtime）
- **排盤引擎**：Swiss Ephemeris（西洋占星）、自行實作八字 / 紫微斗數演算法
- **內容生成**：模板 + LLM 動態填充混合架構

### 內容生成流程

```
命理專家撰寫基礎解讀模板
        ↓
涵蓋各種星座相位 & 八字組合的正向引導語句
        ↓
LLM 根據使用者具體命盤動態組合
        ↓
輸出個人化、流暢、正向的解讀文字
```

## 倫理邊界

- 偵測負面情緒關鍵字時，溫和引導使用者尋求專業心理協助
- App 不扮演治療師角色
- 適時引導專業資源反而提升品牌信任度

## 關鍵成功要素

1. **內容專業度**：需要命理顧問參與，至少在初期模板庫建立階段
2. **AI Prompt 設計**：正向引導的語言品質直接決定產品價值
3. **視覺差異化**：中西融合的獨特美學是最強的傳播武器
4. **信任感建立**：讓使用者感受到 App 真正「懂」命理，而非隨機生成
5. **台灣市場切入**：善用在地命理文化高接受度的優勢，再向華語市場擴展
